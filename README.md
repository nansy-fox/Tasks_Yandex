
# Решение задач на языке Python

## Задание:</a> 
Решите 2 задачи на одном из языков программирования (Python, Groovy, Java, JS, C++, C#), оцените их временную сложность O(n) и опишите работу алгоритма.</a> 
  
### Задача 1:

Найти наибольший общий делитель всех чисел в заданном диапазоне [L, R].

ПРОГРАММНЫЙ КОД:

К задаче 1 код содержится в файле Task1.py, к задаче 2 в файле Task2.py

АЛГОРИТМ ЗАДАЧИ 1:

1. Если L = R, НОД=L=R
2. Иначе НОД диапазона [L, R] равен НОД(L, L+1)

СЛОЖНОСТЬ ЗАДАЧИ 1:

В приложении реализован алгоритм с временной сложностью O(1), так как алгоритм выполняет только проверку условия и время не зависит от размера диапазона.

### Задача 2:

Дан массив целых чисел длины N и целое число K. Разрешена операция удаления первого элемента массива с записью в его конец K-го элемента. Какое минимальное число шагов потребуется для того, чтобы сделать все элементы массива одинаковыми? -1, если это невозможно.

АЛГОРИТМ ЗАДАЧИ 2: 

1. Проверка возможности:

- Если все элементы одинаковы, то 0

- Если все элементы разны, то -1

- Если некорректное K, то -1

2. Нахождение целевого элемента - самого частого в массиве

3. Для каждой позиции целевого элемента вычислить:

- Расстояние до других позиций

- Количество шагов

- Берём максимум по всем позициям

4. Возвращаем минимум из всех вычисленных значений

СЛОЖНОСТЬ ЗАДАЧИ 2:

Реализован алгоритм с линейной временной сложностью O(n), n - размер входного массива данных.
